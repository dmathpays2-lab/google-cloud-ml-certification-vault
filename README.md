## Workflow

### Hermes Integration
Complex automation, custom skill building, and system integration tasks are delegated to **Hermes** — the executor agent. Kimi designs, Hermes builds. Both agents commit to this repository.

### NotebookLM Integration
All study materials from the Google Cloud learning path are synthesized into NotebookLM for:
- **Condensed summaries** of long documentation
- **Q&A generation** for active recall
- **Flashcard creation** for spaced repetition
- **Cross-reference linking** between concepts

### Commit Protocol

| Prefix | Purpose | Example |
|--------|---------|---------|
| `lab-XX` | Course lab completion | `lab-01: completed Vertex AI custom training job` |
| `project` | Portfolio project | `project: deployed agentic pipeline to Cloud Run` |
| `notebooklm` | Study material synthesis | `notebooklm: generated Q&A for module 2` |
| `docs` | Documentation & guides | `docs: updated MLOps architecture diagram` |
| `feat` | New capability | `feat: added automated model retraining trigger` |
| `fix` | Bug correction | `fix: resolved feature store schema mismatch` |
